Calculator Website README


Overview
This is a simple, responsive web-based calculator built with HTML, CSS, and JavaScript. It supports basic arithmetic operations and runs directly in any modern browser without server dependencies.



Features
Basic operations: addition (+), subtraction (-), multiplication (×), division (÷).

Clear (C) and Clear All (CE) buttons for resetting input.

Keyboard support for numbers and operators.

Responsive design for desktop and mobile devices.



Usage
Open index.html in a web browser.

Click buttons or use keyboard to enter numbers and operations.

Press = to compute the result.

Use C to clear current entry or CE to reset everything.



Installation
Clone or download the repository.

No build tools required—open index.html directly.

Host on GitHub Pages, Netlify, or any static file server for online access.



Technologies
HTML5 for structure.

CSS3 for styling and responsiveness.
